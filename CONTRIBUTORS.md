# Contributors

Thank you to everyone who has contributed to dcli! We appreciate your time and effort.

## How to Contribute

We welcome contributions from the community! Please see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on:

- Reporting bugs
- Suggesting features
- Submitting pull requests
- Code style and testing requirements

## Current Contributors

### Core Team
- **Oleg Koval** ([@oleg-koval](https://github.com/oleg-koval)) - Creator and maintainer

## Contribution Guidelines

All contributors agree to follow our [Code of Conduct](CODE_OF_CONDUCT.md) (if applicable).

To be listed here, please:
1. Make a meaningful contribution (code, documentation, bug reports, etc.)
2. Submit a pull request
3. Have your PR merged

We'll add your name and GitHub profile to acknowledge your contribution.

**Want to contribute?** Check out [open issues](https://github.com/oleg-koval/dcli/issues) or [discussions](https://github.com/oleg-koval/dcli/discussions) to get started!
